# **App Name**: LuneCare

## Core Features:

- Language Toggle: Switch between English and French.
- Direct Contact Button: Provide a direct contact button with a phone icon and +1 514 area code.
- Appointment Booking: Allow users to book appointments through the app. Integrate calendar and reminder features.
- Clinic Locator: Enable users to locate Lune clinics nearby using a map interface. The LLM can use its tool to determine location automatically based on user permissions.

## Style Guidelines:

- Background: Off-white/Cream (#FDF8F1) for a soft, clean look.
- Primary Action: Deep Navy Blue (#2B3A67) for main interactive elements like buttons.
- Accent: Soft Lavender/Purple for branding and subtle highlights (#E6E6FA).
- Headers: 'Belleza' (sans-serif) for large, bold lowercase titles like "lune."; Sub-navigation: 'Inter' (sans-serif) for clean, professional text. If the user had requested particular fonts, I would have put the message 'Note: currently only Google Fonts are supported' here.
- Use medical-style icons with a clean, minimalist design. Consider icons that reflect a digital interface (e.g., touch interface) aesthetic.
- Buttons should have a pill shape (high border-radius) with a navy blue background and white text. Incorporate high-tech, medical-style imagery, such as blue-tinted 3D facial meshes.
- Subtle transitions and animations when loading content or submitting forms to enhance user experience.
